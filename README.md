# dsExamPeer

------- INSTRUCTIONS ------------

To run the program, open 3 separate terminals at the project directory.

If you have MAKE installed (otherwise, see below)

    Run the following commands, one in each terminal:

        make client0

        make client1

        make client2

If you don't have MAKE:

    Run the following commands, one in each terminal:

        go run main.go 0

        go run main.go 1

        go run main.go 2
